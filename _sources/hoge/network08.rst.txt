*書籍を買う前に*

| 3minutes Networking
| 
| http://www5e.biglobe.ne.jp/aji/3min/
|
| 文献としては古くなってしまうが、ネットワークの
| 基本を学ぶには良き教材。
| アニメーションがAdobe Flashで作られていたため、
| 今となっては見られなくなっているものの、隙間の時間で
| 少しずつ勉強するのには最適。

| IT用語辞典 e-Words
|
| https://e-words.jp/
|
| ネットワークに限らず、IT用語を網羅した、まさに
| IT用語の百科事典。

| IIJ Bootcamp
|
| https://iij.github.io/bootcamp/
|
| ネットワークには関係ない部分がメインだが、
| エンジニアとして高みを目指すなら、学んでおいて
| 損はない資料集。
| ※ 実は講師も全部を網羅出来てはいないので勉強中
